# GeoSight Competition Submission Package

This folder is the submission handoff for the GeoSight competition build. It is written for a live judged demo first, with backup assets and concise reviewer guidance.

## Contents

- [`demo-scripts.md`](demo-scripts.md): 90-second, 3-minute, and 5-minute walkthrough scripts
- [`pitch-deck-outline.md`](pitch-deck-outline.md): slide-by-slide outline for a short competition deck
- [`methodology-source-sheet.md`](methodology-source-sheet.md): one-page trust and provenance summary
- [`recorded-demo-guide.md`](recorded-demo-guide.md): fallback recording plan if the live demo needs backup
- [`capture-checklist.md`](capture-checklist.md): exact screenshot and GIF targets to record

## Recommended Demo Order

1. Columbia River infrastructure siting
2. Tokyo commercial market scan
3. Washington residential due diligence

## Submission Principles

- Lead with the live demo, not the deck.
- Keep the Columbia River story as the default first impression.
- Show provenance inline on every outcome the judge sees.
- Use the backup artifacts only if the live path misbehaves.
