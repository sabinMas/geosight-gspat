# Lens Usability Audit — Trial & New Feature Badges

**Purpose:** Quickly test all 9 Explorer mode lenses and identify which ones need:
- **Trial Badge** — Known issues, incomplete features, or experimental UI still being refined
- **New Feature Badge** — Newly launched, working well, but users should know it's fresh

**Test Site:** Use the same location (e.g., Phoenix, AZ or your area) across all 9 lenses for consistency.

## Quick Assessment Guide

| Badge | Apply When |
|-------|-----------|
| **Trial** | Feature has known issues, incomplete data, or UI still being refined. Users should expect rough edges. |
| **New** | Feature just launched, working well, but fresh enough that users might not know it exists. |
| **None** | Feature is stable, complete, and established. No badge needed. |
